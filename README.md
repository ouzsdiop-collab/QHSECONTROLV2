# QHSE Control V2

QHSE Control V2 est un **nouveau SaaS QHSE premium** construit from scratch pour offrir une expérience à la fois stratégique, opérationnelle et terrain.

## Vision produit
- Expérience moderne avec **mode clair / sombre**.
- **Mode Essentiel guidé** pour utilisateurs non experts QHSE.
- **Mode Expert** pour analyses avancées, audits et conformité ISO.
- **Mode Terrain** mobile-first pour déclarations rapides et exécution opérationnelle.
- **Mode Admin** pour la gouvernance SaaS multi-entreprises.

## Capacités cibles
- Import **Excel/PDF** robuste avec pré-remplissage et **validation humaine obligatoire**.
- Génération de **rapports PDF niveau cabinet** (et non simple export écran).
- Préparation et pilotage des démarches **ISO 9001, ISO 14001, ISO 45001**.
- Bases réglementaires **paramétrables** selon contexte client.
- Architecture technique **modulaire** pour croissance long terme.

## État du dépôt
- Le nouveau produit est préparé dans `/app`.
- Les références de l’ancien produit sont isolées dans `/reference`.
- **Aucun déploiement n’est configuré à ce stade.**
- **Aucune dépendance n’est installée à ce stade.**
