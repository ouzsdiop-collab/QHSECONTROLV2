# Référence QHSE Control V1

Ce dossier contient des artefacts de l’ancien produit **QHSE Control V1**.

## Règles d’utilisation
- Usage strictement limité à la **lecture** et à l’**analyse**.
- Le contenu de `/reference` ne doit **jamais** être déployé.
- Le contenu de `/reference` ne doit **pas** servir de base active de développement.
- L’archive V1 sert uniquement à :
  - comprendre les fonctionnalités existantes ;
  - inventorier les modules métier ;
  - analyser les parcours utilisateurs ;
  - identifier les modèles de données historiques ;
  - recenser les erreurs et anti-patterns à ne pas reproduire dans V2.

## Structure
- `/reference/archives` : archives brutes (ZIP source V1).
- `/reference/extracted` : zone optionnelle d’extraction temporaire pour analyse future.

## Statut actuel
- L’archive V1 est conservée telle quelle dans `/reference/archives`.
- **Aucune extraction n’est effectuée à ce stade.**
