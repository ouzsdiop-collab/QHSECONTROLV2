# Product Brief — QHSE Control V2

## Positionnement
QHSE Control V2 est un SaaS QHSE premium orienté performance, conformité et adoption métier.

## Personae cibles
- Direction / managers / RH / chefs d’équipe (Mode Essentiel).
- Responsables QHSE / consultants / auditeurs (Mode Expert).
- Opérateurs terrain / superviseurs (Mode Terrain).
- Admins de plateforme et support SaaS (Mode Admin).

## Proposition de valeur
- Un socle de données unique pour tous les modes.
- Une lecture différente selon niveau de maturité utilisateur.
- Une chaîne complète : collecte terrain → analyse → décision → reporting cabinet.

## Contraintes de phase actuelle
- Pas de déploiement.
- Pas de secrets.
- Pas d’implémentation applicative complète.
- Cadrage documentaire et architecture cible uniquement.
