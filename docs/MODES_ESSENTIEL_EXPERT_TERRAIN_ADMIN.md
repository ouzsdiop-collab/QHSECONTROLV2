# Modes produit — Essentiel, Expert, Terrain, Admin

## Mode Essentiel
- Ce n’est ni une version gratuite ni une version dégradée.
- C’est un mode guidé pour dirigeants, managers, RH, chefs d’équipe et non-experts QHSE.
- Il repose sur les mêmes données que le mode Expert.
- Il sert au **pilotage simplifié** : priorités, risques et décisions à prendre.
- Il traduit les indicateurs en langage clair et explique pourquoi ils sont bons ou mauvais.
- Il propose des **actions suivantes recommandées** pour les non-experts.

## Mode Expert
- Cible : consultants QHSE, responsables QHSE, auditeurs, entreprises en démarche ISO.
- Donne accès aux détails techniques/réglementaires/ISO.
- Inclut preuves, écarts, matrices, audits et cotations.

## Mode Terrain
- C’est une **interface mobile simplifiée**, pas une version réduite sans valeur.
- Approche mobile-first avec gros boutons et flux courts orientés action.
- Déclaration rapide d’incident, danger, presque accident et action immédiate.
- Optimisé pour la saisie rapide sur le terrain (objectif < 60 secondes pour un signalement critique).
- Prise en charge photos, signatures, QR codes, commentaires courts (voix possible plus tard).
- Offline-compatible : enregistrement local, synchronisation différée et statuts explicites.
- Actions critiques concentrées : **signaler, consulter, signer, scanner, synchroniser**.
- N’expose pas toute la complexité du mode Expert.
- Statuts de synchronisation visibles : `local_draft`, `pending_sync`, `syncing`, `synced`, `sync_failed`, `conflict_review`.

## Mode Admin
- Gestion SaaS multi-entreprises.
- Gestion utilisateurs, rôles, abonnements, modules activés.
- Pilotage packs réglementaires, logs, imports, exports, erreurs et support.
